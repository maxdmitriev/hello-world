#Setting-up-Git


```

git init
git config user.name "Thomas Anderson";
git config user.email "thomas.anderson@example.com";
git config user.name; git config user.email
git remote add origin https://github.com/maxdmitriev/hello-world.git
git remote -v
git fetch
git checkout master

git status
git add .
git status
git commit -m "First commit"
git push -u origin master


git pull origin master
```
